# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added
- Initial release of @gofindme/client
- `GoFindMeClient` class with API key authentication
- `submitLocation()` method for submitting location updates
- `streamLocations()` method for real-time location streaming via SSE
- `health()` method for checking API health status
- TypeScript type definitions for all API interfaces
- Custom error handling with `GoFindMeError` class
